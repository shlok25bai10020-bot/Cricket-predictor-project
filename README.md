# Cricket-predictor-project
Takes two IPL team as input to play in the match and predicts which Team is going to win it. It is a machine-learning powered predictor which we can use for upcoming IPL matches outcomes that is it predicts the name of the team which is more likely to win and apart from that it also tells the probabilty of the team winning. It uses the basic and fundamentals of AIML (Artifical Intelligence and Machine Learning) which is taught in VITYARTHI. It is built using python, Scikit-Learn, Pandas and it does so by using the dataset which is in form of CSV file and named as ipl_data which contains the data for ipl from 2008 till 2025. The program runs by starting and loading the dataset and then if the dataset exists in the os then it asks for the option to be select whether to predict match(option 1) and exit(option 2)  and afterwards if we select option 1 then, it asks for two teams that is to be selected from available teams for the match and then afterwards it asks for venue and the toss decision taken by the winning team and then it gives the output that is feasible and if no error tells the name of the team which is likely to win and the probability and then afterwards asks for the option to be select whether to predict match(option 1) and exit(option 2). After exiting it shows the output Goodbye!.  

DEMO:
Team 1 (Batting): Chennai Super Kings
Team 2 (Bowling): Mumbai Indians  
Venue: MA Chidambaram Stadium
Toss: bat

 PROJECTED WINNER: Chennai Super Kings
WIN PROBABILITY: 68.4%
